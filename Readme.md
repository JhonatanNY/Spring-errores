# 🚀 Spring Boot - Manejo Global de Excepciones

Este proyecto es una API REST desarrollada con **Spring Boot** que demuestra el manejo centralizado de errores en una aplicación web.

## 📚 Objetivo

El objetivo principal es implementar un sistema de manejo de excepciones global utilizando:

* `@RestControllerAdvice`
* `@ExceptionHandler`
* `ResponseEntity`
* Manejo de errores personalizados

---

## 🧠 Conceptos aplicados

En este proyecto se aplican los siguientes conceptos clave de Spring:

* Inyección de dependencias (`@Autowired`)
* Definición de Beans con `@Configuration`
* Programación orientada a interfaces (`UserService`)
* Uso de `Optional` para evitar `NullPointerException`
* Manejo global de excepciones
* API REST con `@RestController`

## 💡 Aprendizajes clave

* Cómo centralizar el manejo de errores en Spring
* Conceptos de`ResponseEntity` y `@ResponseStatus`
* Uso de `Optional` para evitar errores comunes
* Buenas prácticas en APIs REST
